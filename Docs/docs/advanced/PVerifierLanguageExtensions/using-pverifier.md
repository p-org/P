    TwoPhaseCommit example in Tutorial.

    Clone P Repo locally:
    ```shell
    git clone https://github.com/p-org/P.git
    ```
    Navigate to the TwoPhaseCommit examples folder:
    ```shell
    cd <P cloned folder>/Tutorial/2_TwoPhaseCommit
    ```
